The titular design, having all the necessary components, serves as a prototype which could be built into a functioning app. Thinking up a unique design was challenging, but I crafted rounded and wave-esque assets for a calming atmosphere given its intended usage to study.

Task Tracking App

This project designed with React Native in Javascript allows users to keep track of tasks by adding them to a list and visually striking through them to indicate if it is done. The hardest part in this process was figuring out how to save users' registration data. I ended up using an array of objects which could be iterated through to find a particular user when they log in.
